# 宿題タイマー

宿題時間と休憩時間を交互に知らせる、シンプルなブラウザ用タイマーです。

子どもが自分で区切りをつけながら宿題に取り組めるように、宿題と休憩の時間を大きな表示と色の変化でわかりやすく知らせます。

## できること

* 宿題時間と休憩時間を交互にカウントします。
* 数字をクリックして、スタート・一時停止・再開ができます。
* アイコンをクリックして、宿題時間と休憩時間を切り替えられます。
* 宿題時間と休憩時間は、1分から99分の範囲で変更できます。
* 変更した時間はブラウザに保存され、次回も同じ設定で使えます。
* 時間が終わると音で知らせます。

## 使い方

1. 宿題と休憩の時間を設定します。
2. 数字またはアイコンをクリックしてタイマーを始めます。
3. 宿題時間が終わったら休憩時間へ、休憩時間が終わったら宿題時間へ切り替わります。

## 補足

このアプリは静的なHTMLだけで動作します。設定は使っているブラウザ内に保存され、外部サービスには送信されません。
